what is this.

ateliers.dev は、個人のテクニカルポートフォリオとして、技術的備忘録や教育資材、ガイドライン、ポリシーなどを展示・共有するためのプラットフォームです。

GitHub ガイドライン

『私が GitHub をどうやって運用するか』についてのガイドラインです。 リポジトリの運用方針や、Issue や Pull Request の書き方など、GitHub の使い方について記載しています。

テクニカル ポリシー

コーディングガイドラインや、テスト方針、セキュリティなど、技術的方針について記載しています。 主に、得意とする C# や AI に関することを中心に書いています。

テクニカル ノート

技術的な事のドキュメントです。AI とのやりとりや技術解説の教育資料、技術的なメモを記録し、フィードバックを提供しています。

プロフィール

私のプロフィールです。経歴やスタックスキル、興味などについて記載しています。

FAQ

よくある(かもしれない)質問についてまとめたセクションです。
