# 個人開発ガイドライン

![img](/img/webp/ateliers-dev-development-guidelines.webp)

この配下は「私が普段、個人開発で何をどう判断し、どう使っているか」の参考情報です。

## 構成

### GitHub
GitHub の使い方（branch / CI/CD / repository 管理など）

### Coding
コーディング・設計方針（命名、例外設計、設計原則など）
- C# および .NET 関連

## 読者対象

- 自分
- チームメンバー
- 友人
- この活動に興味のある人

## 位置づけ

このガイドラインは **強制ルールではなく参考情報** です。  
「なぜそうするか」の判断材料として活用してください。
